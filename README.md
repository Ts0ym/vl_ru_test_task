## Запуск проекта

1. **Склонируйте репозиторий**:
   ```bash
   git clone <URL вашего репозитория>
   cd <папка проекта>
   ```

2. **Запустите Docker контейнер**:
   ```bash
   docker-compose up --build
   ```

3. **Откройте браузер** и перейдите по адресу:
   ```
   http://localhost:8080
   ```

4. **Остановка контейнеров** (при необходимости):
   ```bash
   docker-compose down
   ```
